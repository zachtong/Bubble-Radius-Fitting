Reliability & Bug Fixes
Hardened error handling and input validation

ROI Bounds Clamping

ROI coordinates are now clamped to actual image dimensions, preventing array-out-of-bounds errors.

Thread Safety

Result array updates are guaranteed to run on the main thread via Qt's queued signal/slot mechanism. QThread signal naming fixed to prevent crashes on rapid re-trigger.

Per-Frame Error Recovery

Batch worker continues processing even if individual frames fail (corrupted file, detection failure). Errors are logged without aborting the run.

Manual Point Deduplication

Points placed within 5px of an existing point are automatically rejected to prevent near-duplicates that destabilize the Taubin fitter.

Overlay/Wipe Mode Fix

Binary ROI (cropped to ROI dimensions) is now embedded into a full-size canvas before compositing with the original image, preventing shape mismatch errors.

Input Validation

Folder path validation, symlink skipping, MAT file structure verification (safe_loadmat), and invalid-frame filtering before Rmax quadratic fit.

Auto-Tune Search Strategy
Two-phase coarse-to-fine grid search with quality scoring
Parameter Phase 1 — Coarse Phase 2 — Fine
Threshold Grid 0.10 – 0.90, step 0.10 (9 points) ±0.04 around best, step 0.02 (5 points)
Removing Factor Grid 20, 35, 50, 65, 75, 85, 92, 98 (8 points) ±6 around best, step 3 (5 points)
Combinations 72 ~120 (5 candidates × 25 grid)
Top-K Refinement Best 5 coarse results advance to fine phase
Scoring Weighted geometric mean: density0.25 × RMS0.30 × coverage0.30 × plausibility0.15
📊
Fit Quality Scoring
Four-dimensional weighted geometric mean

Edge Density (weight 25%)

Number of detected boundary pixels per pixel of fitted circumference. Target: 0.6 points/px (~1 point every 1.7 px).

RMS Residual (weight 30%)

Root-mean-square distance of edge points from the fitted circle, normalized by radius. Penalized above 15% of radius.

Sector Coverage (weight 30%)

Fraction of 8 angular sectors (45° each) containing at least one edge point. Full coverage = 1.0.

Radius Plausibility (weight 15%)

Binary check: radius must be 3–85% of ROI short side. Outside range → score drops by ~50%.

Good fit ≥ 65%
Fair fit 35–65%
Poor fit < 35%
